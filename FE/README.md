Please follow the following steps to run this application.

1. Add the .env file in root (FE folder)

   # Copy and paste below

   REACT_APP_API_URL=http://localhost:4000
   REACT_APP_API_KEY=your-secret-api-key

   # Ignore source map warnings from node_modules

   GENERATE_SOURCEMAP=false

2. Run "npm i" or "npm install" in terminal to download the node_modules
3. Run "npm start" to run the project.
4. Follow the link: http://localhost:3000
